# Idea Sparring Partner

Idea Sparring Partner is a full-stack AI application where users submit ideas and receive adversarial feedback from four independent AI personas.

## Planned Stack

- Frontend: Angular
- Backend: ASP.NET Core Web API
- Database: Supabase PostgreSQL
- ORM: Entity Framework Core
- AI Provider: Gemini API
- Auth: JWT access token + refresh token
